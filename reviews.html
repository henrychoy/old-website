Review Stuff!

"Review Stuff" is a web app made with Node.js, Express, Handlebars, and MongoDB. It allows users to read and write reviews about movies, TV shows, books, and video games! All data is also available via REST API endpoints in XML, JSON, and HTML formats. Check the GitHub README for more details.


ReviewStuff! - https://reviewstuff.herokuapp.com/reviews
